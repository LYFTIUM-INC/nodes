### üîç **Troubleshooting**

#### **Common Issues:**

1. **Dashboard not loading:**
   - Check if port 8092 is available
   - Verify Python HTTP server is running
   - Check browser console for errors

2. **API not responding:**
   - Verify backend is running on port 8091
   - Check logs: `tail -f logs/mev-backend.log`
   - Restart system: `./stop-mev-system.sh && ./start-ultimate-mev-dashboard.sh`

3. **Exploit feed empty:**
   - Check network connections
   - Verify blockchain nodes are synchronized
   - Check mempool monitoring status
